A cute pixel-art music player for Windows.

How to Download & Install

Click the Download button on sourceforge link
Run the downloaded Bleepy Setup 1.0.0.exe file
Follow the installer steps
Bleepy will appear on your Desktop and Start Menu
Double-click to open and enjoy your music!


How to Use

Drag & drop any audio file onto the player, or click the player area to browse for files
Add multiple songs to build a playlist
Use the shuffle and repeat buttons to control playback
Click anywhere on the progress bar to seek through a song


Supported Formats
MP3, WAV, FLAC, OGG, and most other common audio formats.

Requirements

Windows 10 or newer (64-bit)
